<h1 align="center">DigiMarket - Digital Product Marketplace</h1>

<p>
Pada tugas ini, saya membuat sebuah <b>landing page marketplace digital product</b> dengan nama 
<b>DigiMarket</b> yang bertujuan untuk memudahkan kreator digital dalam memasarkan produk mereka 
serta membantu pengguna mendapatkan berbagai kebutuhan produk digital secara mudah dan cepat.
</p>

<p>
DigiMarket menyediakan berbagai jenis produk digital kreatif seperti:
</p>

<ul>
  <li>E-book</li>
  <li>Template Desain</li>
  <li>Source Code</li>
  <li>UI Kit</li>
  <li>Preset</li>
  <li>Audio</li>
  <li>Video</li>
  <li>Produk digital kreatif lainnya</li>
</ul>

<p>
Website DigiMarket memiliki beberapa halaman dan komponen utama yang dikembangkan, yaitu:
</p>

<ul>
  <li>Navbar / Header</li>
  <li>Hero Section</li>
  <li>Kategori Produk</li>
  <li>Best Seller Product</li>
  <li>Statistik Marketplace</li>
  <li>FAQ (Frequently Asked Questions)</li>
  <li>Footer</li>
</ul>

<h2>🛠 Teknologi yang Digunakan</h2>

<p>
Dalam proses pembuatannya, website ini dikembangkan menggunakan:
</p>

<ul>
  <li><b>HTML5</b> sebagai struktur utama halaman website</li>
  <li><b>CSS3</b> untuk mengatur tampilan dan layout website</li>
  <li><b>Bootstrap Icons</b> untuk penggunaan icon pada beberapa komponen</li>
</ul>

<p>
Website ini menerapkan konsep <b>Responsive Web Design</b> sehingga tampilan dapat menyesuaikan berbagai ukuran layar:
</p>

<ul>
  <li>Desktop</li>
  <li>Tablet</li>
  <li>Smartphone</li>
</ul>
